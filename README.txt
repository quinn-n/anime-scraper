Anime Scraper
Scripts to rip seasons off of anime websites.
Currently only supports wcostream.com
(thewatchcartoonsonline uses it in iframes for hosting, click the "is this video too slow?" button to open the video in wcostream.com)

NOTE: There are 3rd party library requirements, install them with "pip3 install -r requirements.txt"

wcostream.py <save dir> <playlist link>
